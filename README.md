# SportTec

### Plataforma Web para la Gestión de Deportes y Talleres

Desarrollada para la Universidad Tecnológica de Tecámac

</div>

---

## 📌 Descripción General

SportTec es una plataforma web desarrollada para administrar los deportes y talleres ofrecidos por la Universidad Tecnológica de Tecámac.

La aplicación permite gestionar alumnos, maestros, disciplinas deportivas, talleres, horarios e inscripciones, centralizando toda la información en un único sistema.

---

## 🎯 Objetivo

Optimizar y digitalizar los procesos administrativos relacionados con las actividades deportivas y culturales de la universidad, facilitando la gestión de información y mejorando la experiencia de los usuarios.

## 🏗️ Arquitectura

<p align="center">
<img src="https://img.shields.io/badge/Arquitectura-MVC-blue?style=for-the-badge">
</p>

El proyecto fue desarrollado bajo el patrón de diseño **MVC (Modelo - Vista - Controlador)**, permitiendo una mejor organización del código, escalabilidad y facilidad de mantenimiento.

### Componentes

| Capa | Tecnología |
|--------|------------|
| Vista | HTML, CSS, Bootstrap, JavaScript |
| Controlador | PHP |
| Modelo | PostgreSQL |
| Servidor | XAMPP |

## 🔄 Desarrollo del Proyecto

### Análisis

Identificación de los procesos administrativos relacionados con deportes y talleres.

### Diseño

Modelado de la base de datos relacional y definición de los módulos del sistema.

### Implementación

Desarrollo de la plataforma utilizando PHP, Bootstrap y PostgreSQL bajo arquitectura MVC.

Validación funcional de cada módulo para garantizar la integridad de la información y el correcto funcionamiento del sistema.
