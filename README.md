# Resumen Ejecutivo

**SportTec** es una plataforma web desarrollada para la **Universidad Tecnológica de Tecámac**, diseñada para optimizar la administración de deportes y talleres institucionales.

La solución centraliza la gestión de alumnos, docentes, disciplinas deportivas, talleres, horarios e inscripciones, facilitando el control de la información mediante una plataforma moderna, intuitiva y eficiente.

Este proyecto fue desarrollado aplicando principios de **Ingeniería de Software**, arquitectura **MVC (Modelo - Vista - Controlador)** y bases de datos relacionales para garantizar escalabilidad, organización y mantenimiento del sistema.

# Problemática

Antes del desarrollo de SportTec, la administración de deportes y talleres se realizaba de forma manual, lo que generaba:

- Dificultades en el control de inscripciones.
- Organización limitada de horarios.
- Procesos administrativos lentos.
- Duplicidad de información.
- Falta de centralización de datos.
- Baja difusión de actividades deportivas y culturales.

Estas limitaciones impactaban directamente en la gestión institucional y en la experiencia de los estudiantes.

---

# Solución Implementada

SportTec fue desarrollado para digitalizar y automatizar los procesos administrativos relacionados con la gestión deportiva y cultural de la universidad.

La plataforma permite:

- Administración de alumnos.
- Administración de docentes.
- Gestión de deportes.
- Gestión de talleres.
- Administración de horarios.
- Control de inscripciones.
- Gestión de usuarios.
- Administración de roles.
- Consulta de actividades.
- Generación de reportes.

---

# Arquitectura del Sistema

<div align="center">

## MVC (Modelo - Vista - Controlador)

</div>

El proyecto fue desarrollado utilizando la arquitectura MVC, permitiendo una clara separación entre la lógica de negocio, la interfaz de usuario y el acceso a datos.

### Componentes

| Capa | Tecnologías |
|--------|------------|
| Vista | HTML5, CSS3, Bootstrap, JavaScript |
| Controlador | PHP |
| Modelo | PostgreSQL |
| Servidor | XAMPP |

### Beneficios Implementados

- Mayor organización del código.
- Fácil mantenimiento.
- Escalabilidad.
- Reutilización de componentes.
- Separación de responsabilidades.
